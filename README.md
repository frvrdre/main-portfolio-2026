This was orignally supposed to be a practice portfolio used to help me improve my css and html skills however, I ended up getting sinked into the design and ended up making it my main portfolio.

This website is not meant to pretend I already know everything. It is a place where I can show what I am learning, what I am building, and how my skills improve over time.

## About the project

I created this portfolio using plain HTML, CSS and JavaScript so I could strengthen my understanding of the fundamentals before relying too heavily on frameworks.

The design uses a dark developer-inspired theme with blue accents, an animated grid background, responsive project cards and a floating personal logo.

The main goal is to keep improving the website alongside my own development journey.

## Running the project locally

Clone the repository:

```bash
git clone https://github.com/frvrdre/YOUR-REPOSITORY-NAME.git
```

Open the project folder:

```bash
cd YOUR-REPOSITORY-NAME
```

Then open `index.html` in your browser, or use a local development extension such as Live Server in VS Code.
